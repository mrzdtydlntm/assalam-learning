# SMK As-Salam Workshop

This is SMK As-Salam Workshop API

### Tech Stack:

1. App
    - go v1.22
    - nodemon v3.0.3
2. Dependencies
    - labstack/echo/v4 v4.10
    - joho/godotenv v1.5
    - jmoiron/sqlx v1.3.5
3. Database
    - MySQL v8

## Local Development

- Create and setup env file
  ```
  $ cp .env.example .env
  ```
- Instal dependency go
  ```bash
  go mod tidy
  ```
- Run app
  ```bash
  go run main.go
  ```
